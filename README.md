# Análisis de Ventas de Videojuegos

Este proyecto analiza un conjunto de datos de ventas de videojuegos para identificar tendencias, patrones y insights relevantes en la industria de los videojuegos.

## Descripción

Este análisis explora datos históricos de ventas de videojuegos, incluyendo información sobre:
- Ventas por región
- Géneros más populares
- Plataformas de juego
- Puntuaciones de usuarios y críticos
- Tendencias a lo largo del tiempo

## Tecnologías Utilizadas

- Python
- Pandas para análisis de datos
- Seaborn y Matplotlib para visualizaciones
- Jupyter Notebook para el análisis interactivo

## Estructura del Proyecto

- `Analisis de ventas de videojuegos Ice.ipynb`: Notebook principal con el análisis
- `games.csv`: Dataset con la información de ventas de videojuegos
- `.gitignore`: Archivo para excluir archivos innecesarios del control de versiones

## Cómo Ejecutar

1. Clona el repositorio
2. Asegúrate de tener instaladas las dependencias necesarias:
   ```
   pip install pandas numpy matplotlib seaborn
   ```
3. Abre el notebook en Jupyter Lab o VS Code
4. Ejecuta las celdas en orden
